# §6.2 false-negative 검증용 분배형 천정 종목 (VD-10)

> §6.2 topping force-ignore 게이트의 false-negative 검증: 실제 Stage 3→4 천정을
> §6.2 가 잡는지. **non-climactic(분배형) 필수** — climax 형이면 §6.1 이 먼저 발화해
> §6.2 미검증. 데이터 추출(천정 직전 3주 상승 <25% = 비-parabolic) + minervini 통과
> 커버리지 확인으로 선정.

| 우선 | 티커 | 이름 | 천정주 | 천정직전 3주 최대상승 | 천정후 최대낙폭 | 천정±기간 minervini 통과주 |
|---|---|---|---|---|---|---|
| 1 | 053350 | 이니텍 | 2025-06-20 | 12.5% (비-climax ✓) | -19% | 11 |
| 2 | 018290 | 브이티 | 2025-06-05 | 6.0% (비-climax ✓) | -31% | 6 |

선정 근거:
- 둘 다 천정 직전 3주 상승 <25% → parabolic 아님 → §6.1 climax 미발화 → §6.2 단독 검증 가능.
- 이니텍: minervini 통과 11주(최다) — 백필이 천정 전환 구간을 충분히 분석. 1순위.
- 브이티: 천정후 -31%(최대 낙폭) — 가장 명확한 Stage 3→4. 2순위(상보적).

## 백필 검증 범위 (VD-11)

각 종목 천정 전후를 백필 (천정 −8주 ~ 천정 +12주 권장)해 §6.2 발화 확인:
- 이니텍: `--start=2025-04-26 --end=2025-09-13 --tickers=053350`
- 브이티: `--start=2025-04-12 --end=2025-08-30 --tickers=018290`

검증 기록 항목(정직 보고):
- §6.2 가 천정 구간을 force-ignore(topping_distribution) 로 잡았는가
- **발화한 트리거(T-A/B/C/D)가 무엇인가** — reasoning 에서 확인
- **미발화 트리거는 "이 케이스로 미검증" 으로 정직 표기** (2종목으로 4트리거 전부 커버 못 할 수 있음)
- 천정 *이전* 정상 상승 구간(아직 10주선 위)은 §6.2 가 침묵하는가 (G0 false-positive 재확인)

## non-climactic 재확인 (백필 전, 2026-06-13)

> 방식 확정: "직전 3주" 가 아니라 **피크 포함 접근구간(~8주)의 rolling max(1w,2w,3w)**
> (피크주에 끝나는 급등 누락 방지). 근거: §6.1 은 P1∧P2 모두 요구, P2(1–3주 ≥25%)가
> 필요조건 → 피크 접근에서 rolling max <25% 면 §6.1 발화 불가(트리거·성숙도 무관) →
> §6.2 가 유일 기제. weekly_prices.adj_close 로 산출.

| 종목 | 천정주 | 피크 접근 rolling max(1w,2w,3w) | 판정 |
|---|---|---|---|
| 018290 브이티 | 2025-06-05 | 17.3% (05-02, 3w) | <25% → §6.1 비발화 ✓ |
| 053350 이니텍 | 2025-06-20 | 19.5% (06-20, 2w) | <25% → §6.1 비발화 ✓ |

→ 둘 다 §6.1 비간섭 확인, **종목 교체 불필요**. 발화여부는 백필이 판정(non-climactic
필터만 통과시킴 — "pass" 만들려 교체 금지).

**이니텍 주의(기록용, 교체 사유 아님)**: 2025-03-28 에 3주 +88% 의 진짜 parabolic 존재
(초기 advance). 단 백필 범위(04-26~) *이전*에 종료됐고, 6월 천정의 active pace 는 19.5%
(<25%) 이며 entire-advance 최급등은 3월이라 6월엔 P2 해당 불가 → 천정에서 §6.1 발화 불가.
즉 "3월 climax(범위 밖) → 6월 분배형 천정" 구조라 §6.1 과발화 없이 §6.2 단독 검증에 적합.

## #3 (left-censoring) 점검

두 종목 모두 advance 시작(직전 Stage1→2 전환)이 2024년 내 → 104주 창 안 → §6.1 P2
left-censoring 미해당(정상 anchor). 별도 left-censored 케이스 검증은 이번 범위 외
(SK하이닉스·이 2종목 모두 in-window) — 후속 필요 시 >2년 상승 종목으로.
