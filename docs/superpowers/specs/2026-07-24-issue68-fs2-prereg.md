## 1. 등록 동기 (4단계 관찰 결과 — 수치 확정분)

- 4단계 탐색(`2026-07-22-issue68-stage4-exploration.md`): 승격 0건. 최근접 =
  **F-S1(매출 YoY ≥+25%) +7.6%p** — 기준 +8.0%p 에 0.4%p 미달.
- F-S1 은 오닐 매출 조건의 **절반 구현**이었다(모문서 §8 gap 2-1 등록분):
  원문은 "최신 분기 매출 ≥+25% **또는** 최근 3분기 매출 증가율 가속"의
  이중 지선 [HMMS C장]. 미구현 지선(가속)을 채운 완전형이 book-mandated
  후보로 남아 있어, 이를 **신규 후보 F-S2** 로 등록한다.
- 주의(정직성): F-S1 의 +7.6%p 를 본 뒤 그 계열을 재등록하는 것이므로
  데이터 주도 선택이다 — 그래서 채택이 아니라 **A+B 관찰(look 가산) →
  표본 C 판정**의 2단 경로를 강제한다.

## 2. F-S2 정의 (잠금 대상)

- **F-S2** = 지선 ① 최신 분기 `revenue` YoY ≥ **+25%** (모문서 F-S1 그대로)
  **OR** 지선 ② **3분기 연속 가속**: g_t > g_{t−1q} > g_{t−2q}
  (최근 3개 분기 각각의 YoY 성장률이 단조 증가 — 4Q 파생 포함).
- 지선 ② 는 g 3개가 전부 정의(기준값 > 0)될 때만 평가 — 하나라도
  턴어라운드/미정의면 지선 ② 판정불가.
- 라벨 합성: 지선 중 하나라도 pass → **pass** / 둘 다 fail → **fail** /
  지선 ① turnaround 이고 지선 ② 가 pass 아님 → **turnaround** /
  그 외(판정불가 혼재) → **indeterminate**.
- 데이터 경로·가드: 모문서 §2 그대로(매출은 금액 기반 — 주식수 가드 비적용).
  +25% 임계·모든 공통 규칙 변경 없음.

## 4. 표본 C 의존성 (선행 작업 — 판정과 분리)

- 표본 C 트레이드 셋은 **#52 본실행**(LLM 분류 백필, 별도 비용 승인 트랙)의
  산출물 — 본 트랙은 그 완료를 기다린다(트레이드 생성에 관여하지 않음).
- DART 실적: 표본 C 100종목 백필 필요 + **연도 하한 2016 연장**(C 구간
  2017-H2~2020 의 지선 ① YoY 기저 확보 — 무료 API ~5,000콜). eps-published
  패스 포함. 이 작업은 트레이드·성과와 결합되지 않으므로 판정 전 수행
  가능(데이터 준비).
- **알려진 한계(실측, 2026-07-24)**: DART 주요계정 **분기** 데이터는 2016년이
  하한 — 2015년 분기는 API 자체에 없음(2종목 교차 실측: 2015 분기 status
  013 · 2016 분기 000, 연간은 2015 존재). 따라서 표본 C 최초기(2017-H2 진입,
  최신 분기가 2017 초·Q4 2016 파생이 끼는 구간)의 **지선 ② 는 YoY 기저(2015
  분기) 부재로 구조적 판정불가**가 된다. 지선 ① 은 2016 연장으로 충분.
  이 판정불가는 모문서 §2 원칙대로 제3그룹 처리(통과/탈락 오염 없음)하며,
  판정불가 ≤40% 기준 계산에 그대로 포함한다(완화 없음).
- 해석 가드 상속: 관찰·판정 각각 귀무 하 통과 확률이 낮지 않음(모문서 §4) —
  표본 C 재현까지 가야 증거로 취급한다.
