## 2. 공통 정의 (모든 필터가 공유)

- **데이터 접근**: `get_financials_asof(ticker, as_of=진입일)` 만 사용(strict <,
  T+1 가용 — look-ahead 방지 chokepoint). 직접 테이블 조회 금지.
- **최신 분기**: as-of 가시 행 중 fiscal_end 최신 행을 취해 — 분기
  행(11013/11012/11014)이면 그 **3개월 단독** `thstrm_amount`(2단계 §3 실측
  의미론 — fiscal 라벨은 누적 기간이므로 합산·기간 계산에 사용 금지),
  연간(11011)이면 아래 4Q 단독 파생. 오래된 분기로 대체하지 않음.
- **4Q 단독 파생**: 4Q 단독 = 연간 − (1Q+2Q+3Q).
  같은 연도 4개 행이 전부 status=ok·같은 fs_div 일 때만 — 아니면 판정불가.
  (전년 동기 4Q 도 동일 규칙으로 파생.)
- **YoY 성장률**: g = (X_t − X_{t−1y}) / |X_{t−1y}|. 전년 동기 = 같은
  reprt_code · bsns_year−1 (비12월 결산도 reprt 기준 — fiscal 라벨 불신).
- **흑자전환 규칙**: 기준값(전년 동기) ≤ 0 이면 % 미정의 —
  - 현재 > 0 (적자→흑자): **통과**로 간주 + `turnaround` 태그로 분리 집계
    (오닐의 턴어라운드 수용 반영, 민감도 확인 가능하게)
  - 현재 ≤ 0 (적자 지속·적자 전환): 탈락
- **주식수 이상치 가드** (F-C1·F-C2 만): YoY 비교쌍의 두 연도 `shares_outstanding`
  비율 ≥100 또는 ≤0.01 → 해당 판정 **판정불가** (근거: 054090 의 2023
  주식총수가 DART 원천에서 1000× 오기재 — §5 실측).
- **판정불가(제3그룹)**: 위 가드 발동, as-of 가시 행 부족(전년 동기 부재 등),
  필요한 값 NULL, 4Q 파생 불가. 판정불가는 통과/탈락 어느 쪽에도 섞지 않고
  별도 그룹으로 집계(조용한 편향 방지).

## 3. 잠금 대상 필터 정의 3개

| id | 정의 | 원문 근거 |
|---|---|---|
| **F-C1** | 최신 분기 `eps_derived` YoY ≥ **+25%** (또는 turnaround) | 오닐 C 최소 기준 |
| **F-C2** | F-C1 **AND** 가속: g_t > g_{t−1q} (직전 분기의 YoY 대비 개선) | 오닐 C 의 가속 선호 |
| **F-C3** | 최신 분기 `operating_income` YoY ≥ **+25%** (또는 turnaround) | **원문에 없음** — 파생 EPS 오차 헤지용 번안(주식수 나눗셈 무관) |

- F-C2 의 g_{t−1q} = 최신 분기의 직전 분기(달력상 한 분기 전, 4Q 파생 포함)의
  YoY. 둘 중 하나라도 흑자전환/미정의면 F-C2 는 **판정불가**(통과 아님 — 보수).
- +25% 는 오닐 C 의 최소 기준. `thresholds.py` SSOT 편입은 실전 채택 시점으로
  미룸(YAGNI) — 탐색·판정 단계에서는 본 문서가 정의의 권위.
