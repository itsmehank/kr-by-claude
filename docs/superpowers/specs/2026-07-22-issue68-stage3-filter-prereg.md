## 2. 공통 정의 (모든 필터가 공유)

- **데이터 접근**: `get_financials_asof(ticker, as_of=진입일)` 만 사용(strict <,
  T+1 가용 — look-ahead 방지 chokepoint). 직접 테이블 조회 금지.
- **최신 분기**: as-of 가시 행 중 fiscal_end 최신 행. 분기 행(11013/11012/11014)
  이면 3개월 단독 값(2단계 §3 실측 의미론 — fiscal 라벨은 누적 기간이므로
  합산·기간 계산 사용 금지), 연간(11011)이면 아래 4Q 단독 파생. 오래된 분기로
  대체하지 않음.
- **EPS YoY 의 입력 우선순위** (F-C1·F-C2):
  1. **공시 쌍**: 같은 공시의 (`eps_published`, `eps_published_prior`) —
     전체계정 API 기본주당이익의 당기(3개월 단독)와 전년 동기(분기=frmtrm_q,
     연간=frmtrm). 같은 공시의 비교값은 소급 재작성 값이라 **무상증자·액면분할·
     지배/전체 혼재·자기주식·주식수 근사 문제에 구조적으로 면역**(실측 016800).
  2. **파생 폴백**: 공시 쌍 결측 시 `eps_derived` 의 연도 간 쌍 —
     `eps_fallback` 태그 병기 + 주식수 2단 가드 적용(아래).
  3. 둘 다 불가 → 판정불가.
- **4Q 단독 파생**: 4Q = 연간 − (1Q+2Q+3Q), 순이익(파생 경로)·영업이익·매출에
  적용. 같은 연도 4개 행 전부 status=ok·같은 fs_div 일 때만 — 아니면 판정불가.
  전년 동기 4Q 도 동일 규칙. 공시 EPS 는 4Q 단독이 공시되지 않으므로 4Q 시점의
  F-C1/C2 는 항상 파생 폴백 경로 — `q4_derived` 태그 병기, 탐색 집계에서 제외
  민감도 예약(정정·연말 조정이 4Q 파생값에 집중 흡수되는 함정 — 외부 검토 3-2).
- **YoY 성장률**: g = (X_t − X_{t−1y}) / |X_{t−1y}|. 파생·영업이익·매출 경로의
  전년 동기 = 같은 reprt_code · bsns_year−1 (비12월 결산도 reprt 기준 — fiscal
  라벨 불신). 공시 쌍 경로는 같은 공시 내 값이므로 매칭 불요.
- **결산기 변경 이행기**: 연간 행의 회계기간(fiscal_start~end)이 12개월 ±20일
  밖이면 그 연도가 낀 YoY 는 판정불가(외부 검토 3-2b).
- **턴어라운드(제4 라벨)**: 기준값(전년 동기) ≤ 0 이고 현재 > 0 이면 통과가
  아니라 **`turnaround` 라벨**. 1차 집계는 pass(ex-turnaround) 기준, 턴어라운드
  합산은 민감도 전용(외부 검토 차단① — 오닐의 턴어라운드는 강도 조건부 수용
  [HMMS 4장]이므로 자동 통과는 원문 불일치. 강도 조건 구현은 표본 C 단계로
  이관. 본 규칙 태그: design-judgment). 기준값 ≤ 0 이고 현재 ≤ 0 은 탈락.
- **주식수 2단 가드** (파생 폴백 쌍에만): 비교 두 연도 `shares_outstanding` 비율
  r 에 대해 — r ≥ 100 또는 ≤ 0.01 = **데이터 오류** 판정불가(근거: 054090 의
  2023 주식총수 DART 원천 1000× 오기재 실측); r ∉ [0.5, 2] = **기업행위
  (무상증자·액면분할·병합) 의심** 판정불가(외부 검토 차단②). 실측 분포(연간
  인접쌍 1,226개): [0.5, 2] 밖 46쌍(3.8%) — 판정불가 비용 수용 가능. 잔여
  한계: 밴드 내 1.02~2× 변동 226쌍(18.4%)의 왜곡(최대 ~2×)은 미차단 — 공시
  쌍 우선 정책이 1차 방어이고, 파생 폴백 잔존분은 known limitation(§8).
- **판정불가(제3그룹)**: 가드 발동, as-of 가시 행 부족, 필요한 값 NULL, 4Q 파생
  불가, 결산기 이행기. 통과/탈락 어느 쪽에도 섞지 않고 별도 집계. 판정불가는
  무작위 결측이 아님(상장 초기 편중 — 오닐 선호 신생주[N]를 구조적으로 배제하는
  recall 편향, 외부 검토 2-4)을 명기하고, 그룹 자체의 성과 통계도 병행 보고.

## 3. 잠금 대상 필터 정의 4개

| id | 정의 | 원문 근거 |
|---|---|---|
| **F-C1** | 최신 분기 EPS YoY ≥ **+25%** (§2 입력 우선순위) | 오닐 C 최소 기준(25~30% 하한 서술 범위 내) |
| **F-C2** | F-C1 **AND** 가속: g_t > g_{t−1q} | 오닐 C 의 가속 선호(다분기 가속의 2점 최소형) |
| **F-C3** | 최신 분기 `operating_income` YoY ≥ **+25%** | **원문에 없음**(design-judgment) — 단 HMMS 3장 "일회성 특별이익 제외" 원칙의 부분 구현(영업외·중단영업손익 면역) + 파생 EPS 오차 헤지(외부 검토 Q1 격상) |
| **F-S1** | 최신 분기 `revenue` YoY ≥ **+25%** | 오닐 C 의 나머지 절반(장 제목 "…and Sales" — book-mandated, 외부 검토 차단③). revenue NULL 86/5,586행(1.5%) = 판정불가 흡수 |

- F-C2 의 g_{t−1q} = 최신 분기의 직전 분기(달력상 한 분기 전, 4Q 파생 포함)의
  YoY. 두 g 중 하나라도 턴어라운드/미정의면 F-C2 판정불가(통과 아님 — 보수).
- F-C3·F-S1 의 YoY 는 금액 기반(주식수 무관 — 가드 비적용), 턴어라운드·판정불가
  규칙은 §2 와 동일.
- +25% 는 오닐 C 최소 기준. `thresholds.py` SSOT 편입은 실전 채택 시점으로
  미룸(YAGNI) — 탐색·판정 단계에서는 본 문서가 정의의 권위.

## 4. 탐색 프로토콜 (4단계 — 잠금 후 실행)

- 모집단: 표본 A+B 풀링 매매 에피소드(기존 백테스트 산출 경로 재사용). 각
  에피소드의 **진입일** 기준으로 F-C1/C2/C3/S1 각각 pass / fail / 판정불가 /
  turnaround 라벨 부여(+`eps_fallback`·`q4_derived` 태그).
- 집계: 필터×그룹별 n·승률·R 분포·풀링 CI(기존 표본 B 분석과 동일 방법) +
  판정불가 그룹 성과 병행 보고 + q4_derived 제외 민감도.
- **실효 비교 횟수 = 8 look**(4 필터 × {1차 pass-ex-turnaround, 턴어라운드 합산
  민감도}) — 판정 단계는 이 수를 전제로 보수 설정(외부 검토 2-2).
  **승격 판정 입력은 이 8 look 으로 한정**하며, q4_derived 제외 민감도·
  판정불가 그룹 성과·입력 채널 구성비(공시쌍/파생 폴백/q4_derived)는
  **기술(descriptive) 집계로서 승격 판정에 입력되지 않는다**(v2 재검토
  차단 B). F-C1⊂F-C2 중첩으로 두 look 은 독립이 아님 — 동시 통과를 독립
  증거 2건으로 해석하지 않는다.
- **해석 가드**(v2 재검토): 귀무(효과 없음, 기저 승률 ~25%) 하에서도 필터당
  통과 확률 ~20%(pass n=25 기준 승률 차 SE ≈ ±9.5%p > +8%p) — **탐색 통과는
  증거가 아니며**, 검정은 표본 C 사전등록의 몫. 풀링 CI 는 한 종목 복수
  에피소드의 클러스터링으로 과소추정 가능(기존 표본 B 분석 방법의 한계 상속).
- **승격 기준(사전등록)** — 필터별 독립 판정, 아래 3개 모두 충족 시에만 표본
  C(#52) 판정 사전등록으로 이관(외부 검토 차단⑤):
  1. pass(ex-turnaround) 군 n ≥ **25**
  2. pass − fail 승률 차 점추정 ≥ **+8%p**
  3. 판정불가 비율 ≤ **40%**
- 임계값·정의는 탐색↔검증 간 **변경 금지** — 변경하면 신규 후보로 취급(look
  수에 가산). **관찰 라벨만 부여, 채택 없음** — 채택 판정은 표본 C 사전등록
  (별도 문서) 몫. 관찰/채택 분리 규율 유지.

## 8. Known limitations / book-fidelity gap 등록

1. **일회성 이익 미조정**: 파생 EPS 분자(당기순이익)는 특별이익 포함 — HMMS
   3장 "일회성 특별이익 제외" 미구현(공시 EPS 도 기본주당이익이라 동일).
   F-C3 이 부분 헤지. [gap]
2. **최근 2분기 요건 미구현**: HMMS 3장 "both of the last two quarters" 권고
   대비 F-C1 은 1분기 최소형. [gap]
2-1. **F-S1 가속 지선 미구현**: 오닐 매출 조건은 "≥25% **또는** 최근 3분기
   증가율 가속"의 이중 지선 — F-S1 은 첫 지선만(책보다 엄격 → recall 편향,
   precision 무해). [gap, v2 재검토]
3. **정정공시 값의 경미한 look-ahead**: 주요계정·전체계정 응답은 최신(정정 후)
   값인데 as-of 는 원공시일 — 정정 전 시점에 정정 후 값이 보임. 버전 이력
   미보관(비용 대비) — known limitation. [limitation]
4. **파생 폴백 쌍의 밴드 내 주식수 변동**(1.02~2×, 18.4%) 왜곡 미차단 —
   공시 쌍 우선이 1차 방어. [limitation]
5. **분기 공시 EPS 커버리지 <100%**: 결측 셀은 파생 폴백(태그) — 비율은
   §5.3 실측 기입. [limitation]
