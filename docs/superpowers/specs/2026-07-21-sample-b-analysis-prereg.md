# 표본 B 아웃오브샘플 분석 — 사전등록 (2026-07-21)

표본 A(동결 100, 2021-2024)에서 나온 판정을 **독립 표본 B**(신규 동결 100종목, 같은 기간,
`frozen_sample_b.py`, seed 20260713)로 재검증한다. 이 문서는 **실행 전 잠금**되며,
판정 기준·상수·시드는 이후 변경하지 않는다.

**잠금 시점 진술**: 이 문서 커밋 시점까지 표본 B 의 판정 지표(P1~P4: entry×국면 교차,
수익 CI, 프리미엄 stopout률)는 **계산하지 않았다**. 지금까지 확인한 것은 진행 모니터링
(분류 분포: watch 1,796 / ignore 142 / entry 14)과 데이터 품질(#49·#50 스윕)뿐이다.

## 2. 판정 (4개 — 1차 기준, 사전등록 잠금)

### P1. 분류층 방어 (§7.1 대칭)
- 지표: R_down = entry-rate(downtrend ∪ correction 국면 분류점),
  R_up = entry-rate(confirmed_uptrend 국면 분류점). 국면 = analyzed_for_date 의
  시장상태(`phases.py`), 엔진 = `entry_rate_by_phase`(표본 B 리스트 필터).
- **기준: R_down / R_up ≤ 0.5 → PASS** (A 와 동일. A 실측 0/1,075 ÷ 1/292 = 0.0).
  R_up = 0 이면 코드 규약대로 **ratio 미정의 → 자동 PASS 없음, 수동 해석**
  (`profitability_run.py` gate_71: pass=False + note).

### P2. 트리거확인층 감사 (개선 1 대칭)
- 대상: 결정론 트레이드 시뮬(보정 후, §3 상수)이 만든 진입 중 진입일 국면 ∈
  {downtrend, correction} 전건. 각 건에 5b LLM 확인(`evaluate_pivot_trigger_v1.md`,
  build_for_5b prior_row 주입, sonnet 핀) 1회 실행.
- **A 와의 의도적 차이**: A 감사는 1차(비보정) 66 트레이드 기준 33건이었으나,
  B 는 **보정 후** 트레이드 기준이다 — 5% 추격 룰이 반영된 실전 정합 집합을
  감사하는 것이 목적에 더 맞다(감사와 보정이 병행 개발됐던 A 의 시점 제약 제거).
- **기준(A prereg 원문 복사): G = go_now 건수 / 대상 건수 (wait·abort = 차단).
  G ≤ 0.5 → 트리거 확인 층의 게이트 방어 입증. G > 0.5 → §3.5 프롬프트 재검토 대상.**
- 기록: `data/backtest/trigger_audit_sample_b_20260721.json` (신규 파일 — A 의
  `trigger_audit_20260702.json` 과 분리). LLM 1회, 재실행 비교 금지. production
  테이블 쓰기 0.

### P3. 수익성 (개선 2·3 대칭)
- **1차 = 표본 B 단독**: 보정 후 트레이드(§3 상수)의 평균 시장초과수익, 종목 클러스터
  부트스트랩 95% CI (BOOT_B=10,000, 시드 **20260721**).
  - **CI 하한 > 0 → "시장 초과 입증"으로 판정 갱신. 0 포함 → 미입증 유지.**
- 보조(판정 아님): ① 풀링 A+B(200종목) CI — 검정력 참고. ② 플라시보 대조군 —
  **표본 B 단독**, 같은 종목·같은 보유 거래일수·진입일만 무작위(2021-2024),
  N=1,000, 시드 20260721. p 값은 참고 보고.

### P4. 추격상한 (5→3% 후보 판정)
- 측정 엔진: **포트폴리오 v2**(pivot 스톱 3층 스택 — I1 과 같은 실행의 트레이드
  로그에서 추출). 진입 프리미엄 구간(0–1% / 1–3% / 3–5%)별 stopout 률·평균 수익.
- **기준: 표본 B 에서 3–5% 구간 stopout 률이 0–3% 합산 구간보다 ≥ 20%p 높으면
  추격상한 5→3% 하향을 "채택 후보"로 승격.**
- **표본 크기 가드: B 의 3–5% 구간 진입이 8건 미만이면 판정 보류** — A+B 합산
  수치를 보조 관찰로 기록하고, 판정은 표본 축적 이후로 미룬다(A 는 4건뿐이었음).
- 승격되어도 `thresholds.py` 반영은 별도 작업(임계 변경 → 의존성 맵 체크리스트 트리거).
