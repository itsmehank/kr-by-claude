## §1. 블라인드 판정 (SEALED 개봉 전 — 이 섹션은 개봉 전 커밋됨)

판정 질문: 각 케이스에서 analyst_A / analyst_B 중 어느 쪽 verdict·reasoning 이
Minervini/O'Neil 방법론(및 이를 인코딩한 A 프롬프트 게이트)에 더 부합하는가.

### case_01 (004020, 2021-02-13) — A: watch / B: ignore(topping)

**판정: B 우세.** 근거: 10-11주 +106% 수직 상승(climax 성격) 직후 주봉 종가가
10주선 하회(G0: 40,950 < 41,345) + 상승 개시 이래 최대 주간 하락(T-A: −13.2%) —
§6.2 topping 게이트의 발화 요건(G0 + T-A~D 중 1)이 성립하고, HMMS p.268-269 의
천장 징후(climax 후 분배)와 정합. RS 89→78 하락 다이버전스도 부합. A 는 고점 후
~5주 시점에 cup 형성 서사로 watch 를 부여했으나, cup 최소 7주(MIN_BASE_WEEKS)
미달 시점 + climax 직후 문맥에서 base 서사는 책 기준 시기상조. B 의 게이트 산술이
명시적이고 정확.

### case_02 (012450, 2022-08-27) — A: ignore(climax) / B: watch(extended) — **참고용, 미계수**

원본 실험에서 `success:false, image_unprocessed_signal`(이미지 처리 실패)로 남은
호출이 포함된 케이스 — "모델 간 판단 차이" 표본으로 부적격이라 판정 집계에서 제외
(브리프 2026-07-12 발견 사항). 참고 소견만: B 의 E1(leadership) 적용은 "적법 base
돌파 후 3주 내 급등" 전제가 필요한데 B 스스로 16개월 횡보를 gate2 기각(wide-loose,
적법 base 아님)했으므로 E1 근거가 자기모순 — 3주 +92% 수직에는 A 의 climax_run 이
책 정합 우세. 단 이 케이스는 집계 밖.

### case_03 (060980, 2021-10-30) — A: watch / B: ignore(topping)

**판정: B 우세.** 근거: 단일주 +23% climax(10배 거래량, 52주 신고가) 후 base 없이
분배 군집, 주봉 종가 10주선 첫 이탈(G0: 50,900 < 51,160) + 25일 분배 5개(T-D ≥4)
— §6.2 발화 성립. RS 음의 다이버전스·거래량 0.25× 고갈 모두 천장 국면 부합. A 는
동일 climax 를 인지하고도 시장 사유 상한(watch)에 머물러 §6.2 force-ignore 를
미적용 — 게이트 인코딩 기준 B 가 정확.

### case_04 (078350, 2022-01-01) — A: ignore(climax) / B: watch(extended)

**판정: A 우세.** 근거: §6.1 게이트 워크스루가 요건 전부를 명시 충족 — P1(anchor
2021-07 이후 18주+ 성숙), P2(3주 롤링 +57% ≥25% 이자 전체 상승 중 최급경사),
T1(최대 주간 스프레드)·T2(최대 주간 거래량 5.4M), 3차+ 후기 base 발이라 E1 면제
불가. 11~12월 +94% 를 base 구조 없이 수직 가속 + SMA200 +108% — 교과서적
blow-off(HMMS p.262-263). B 는 extended_from_ma 로 watch 에 머물러 §6.1
force-ignore 를 미적용.

### case_05 (079960, 2021-01-09) — A: ignore(climax) / B: watch(cup_with_handle, E1)

**판정: B 우세.** 근거: 쟁점은 base 카운트 — A 는 "3~4번째 base"(후기 → climax
성숙 12주 게이트 경로), B 는 "COVID 급락으로 리셋된 이후 1~2번째 base". O'Neil
의 base 카운트는 대세 하락(약세장)을 거치면 리셋된다 — 2020-03 COVID 급락 후
9~12월 cup(depth 18% ≤33%, 선행 상승 ~35% ≥30%)은 리셋 후 초기 base 로 보는
것이 책 정합. 그 위 1/4 pivot 돌파(거래량 1.9×) 후 3주 내 +38% 급등은 §6.1 E1
(초기 base 돌파 직후 급등 = climax 아닌 leadership) 요건에 정확히 해당 — "가장
빠른 상승은 적법 base 돌파 직후에 나온다"(O'Neil). extended 로 진입만 보류한
B 의 watch(extended) 가 규칙 구조와 부합.

### 해석 (패턴 판정)

1. **불일치 계열의 성격**: Sonnet 열세 4건 중 3건은 §6.1/§6.2 force-ignore 미적용
   (climax/topping 을 watch 로 과소 강등), 1건은 base 카운트 리셋 미적용으로 climax
   과잉 적용(정당한 E1 leadership 을 ignore). 즉 climax/topping **경계의 게이트
   산술·규약 적용 정확도** 문제로 수렴 — 원 20건 실험의 "불일치가 인접 1단계
   (watch↔ignore)뿐"이라는 관찰과 정합.
2. **verdict 대체 판단**: 불일치 케이스에서 Sonnet 이 옳았던 사례가 0건이므로,
   **현 프롬프트 상태에서 verdict 층의 Sonnet 대체는 비권장** — 이미지 신뢰성([A]
   PASS)과 별개로, 경계 판정 품질이 하락한다.
3. **경로**: 이 불일치의 원인 축(§6.1/§6.2 정량 게이트, base 카운트)은 이슈 #23 이
   예고한 "§6.1/§6.2 코드 이관"의 대상과 정확히 일치한다 — 게이트 산술이 코드로
   이동하면 이 계열 불일치는 모델 무관하게 구조적으로 소거되므로, **이관 완료 후
   소표본 재평가**가 Sonnet 전환의 합리적 재개 지점.
