## P1. 분류층 방어 — PASS

- 하락·조정기 분류점에서 entry 분류 **정확히 0건** (r_down = 0.0).
- 유일한 entry 편중은 confirmed_uptrend (r_up 0.043 — A 0.003 보다 높음: B 의 entry
  14건 중 13건이 상승 국면. #50 제외 1셀 반영).
- ratio 0.0 ≤ 0.5 → PASS. §3.5 hard rule 이 신규 100종목에서도 동일 작동.

## P2. 트리거확인층 감사 — PASS (G = 0.000)

- 대상: 보정 후 시뮬 93건 중 진입일 국면 ∈ {downtrend, correction} = **43건**
  (A 와의 의도적 차이: A 는 1차 비보정 66건 기준 33건 — prereg P2 명기).
- **43건 전부 wait** — go_now 0건, failures 0. 실전 시스템이라면 하락·조정기 매수
  노출 0 을 재확인. 기록: `data/backtest/trigger_audit_sample_b_20260721.json`
  (LLM 1회, sonnet, 재실행 비교 금지).

## P3. 수익성 — 미입증 유지

- 보정 후(5% 추격·연도별 비용·밴드) **93 트레이드**: 평균 시장초과 **+0.522%**,
  종목 클러스터 부트스트랩 95% CI **[−3.697, +6.725]** (BOOT_B 10,000, seed 20260721)
  → 0 포함 → **미입증 유지**.
- 플라시보(진입일 무작위 N=1,000, B 단독): 실제 +0.522 vs 플라시보 평균 +1.42,
  **p = 0.699** — 돌파 타이밍 가치 미입증 (A p=0.194 보다 약함. 실제가 무작위 진입
  평균보다도 낮음).
- 2025 절단 제외 민감도: 85건 +0.661% — 결론 불변.
- **보조(판정 아님) 풀링 A+B**: 149건, 평균 +1.537%, CI **[−2.944, +7.496]** (seed
  20260721) — 검정력 ~2배에서도 0 포함. "표본을 늘리면 유의해질 것" 가설에 부정적.

## P4. 추격상한 5→3% — hold (승격 기각)

armA-prod(200종목) 트레이드 로그의 진입 프리미엄 구간(pivot 대비):

| 구간 | n | stopout(stop8) | 평균 pnl |
|---|---|---|---|
| 0–1% | 18 | 66.7% | +5.65% |
| 1–3% | 11 | 45.5% | −1.26% |
| 3–5% | 9 | 55.6% | −3.95% |

- 격차 = 55.6 − 58.6(0–3 합산) = **−3.0%p** < +20%p → **hold**.
- **A 의 "3–5% 구간 stopout 100%(4/4)" 는 소표본 노이즈였음이 드러남** — n=9 로
  늘리자 프리미엄 구간의 stopout 이 특별히 높지 않다(평균 pnl 은 여전히 최악 −3.95%
  — 수익성 관점의 열위는 남지만, 사전등록 판정 지표는 stopout 격차).
- 결론: 추격상한 5→3% 하향의 근거 없음. thresholds.py 무접촉.
