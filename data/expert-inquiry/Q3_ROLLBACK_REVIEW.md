## 1. Q3 설계 — 무엇을, 어떻게 고쳤나 (상세)

### 1-1. strobing 의 기계적 원인
기존 §6.1 은 매주 독립 판정 + 트리거가 "전체 advance 이래 최대 고저폭(T1)/최대 거래량(T2)"
이라, 신기록을 세운 주에만 발화. 연속 parabolic 이라도 비갱신 주(소폭 눌림·약한 상승)는
트리거 미발화 → watch 로 깜빡임.

### 1-2. stateless 제약 (핵심 — 설계를 좌우)
백필/주말분류는 **각 주를 독립 분석**한다(LLM 은 그 주 시점 데이터만 보고, 이전 주의 분류
결과를 모름). 따라서 "지난주 climax 였으니 이번주도 유지" 같은 **상태 의존 규칙은 불가**.
연속성은 반드시 **그 주 차트 자체에서 추론 가능**해야 한다.

### 1-3. 연속성 규칙 (차트추론형)
"blow-off 는 연속 상태, per-week 신기록 체크 아님. terminal acceleration 이 확립
(P1+P2+≥1 트리거, 즉 전체 advance 최급등 1~3주 페이스가 **최근 4주 내** 발생)되면, 신기록
비갱신 주라도 blow-off 내부면 climax_run 지속 발화. 중간 눌림·소폭 신고가도 climax(베이스 아님)."

### 1-4. 이중 EXIT + 앵커 선택 (이 부분이 가장 중요)
EXIT(post-climax 전환, 미발화) 조건 = **둘 중 하나**:
- (a) 고점 대비 >15% 조정, 또는
- (b) **최급등 주(P2 peak)** 후 >4주 경과.

**앵커를 "최급등 주"로 잡은 이유 (가격 고점 앵커면 베이스 누출):**
SK하이닉스 가을 예시 (수정 종가):
```
최급등 주(3주 페이스 최대) = 10-17 (+38.3%/3주)
가격 고점               = 11-07 (580,000원)
post-climax 베이스 시작  = 11-21~ (521,000, -10.2%)
```
- **최급등주(10-17) 앵커 + 4주** → 11-14 까지 ignore, **11-21 EXIT → 베이스 watch**. ✓
- **가격고점(11-07) 앵커 + 4주** → 12-05 까지 ignore → **11-22·11-29 베이스가 ignore 로 누출** ✗
  (= watch-기아 회귀). 게다가 11-21 은 -10.2% 로 15% EXIT 도 아직 미발동 → 베이스 4주 누출.

→ 4주 시계를 *최급등 주*에 앵커해야 "늦은 소폭 신고가가 blow-off 를 베이스 구간으로 끌고
가는 누출"을 막는다. 이 설계가 안전게이트의 핵심이었음.

## 5. 🔴 방법론 발견 — 백필이 LLM 비결정적 (가장 중요)

**temporal-scope(연속성)만 고쳤는데, 그와 무관해야 할 개시판정(09-20/10-18/04-25)·E1
(07-05)·베이스(01-24)까지 흔들렸다.** 이는 변경의 인과 효과가 아니라 상당 부분 **LLM
샘플링 비결정성(noise)** 임을 시사한다.

함의:
1. **단일 run before/after 는 prompt 변경 효과와 노이즈를 분리 못 한다.** diff 7주 중
   "확실히 Q3 때문"이라 단정할 수 있는 건 거의 없다(strobe 메움 11-01/05-02 조차 노이즈
   가능성 배제 불가).
2. 모든 백필 기반 결론에 **노이즈 마진**이 있다. (false-positive PASS 의 핵심 정성 결론
   — 베이스 20주 watch, 표준피벗 watch — 은 견고하나, 주 단위 정밀 카운트는 흔들릴 수 있음.)
3. 결정적 효과 측정엔 **다회 run + 다수결/통계**, 또는 temperature 통제 등이 필요(비용 큼).

- → **옵션 B(API 이전 → temp=0) 철회.** Claude 모델로 결정론-via-temp 는 불가능.

**★ 멱등성 재프레이밍이 진짜 해법**: 비결정성은 **삭제+재실행(비교)에서만** 발생.
정상 운영(1회 백필 → classification_backfill 저장 → 분석)은 안정. 운영 모델 =
"1회 → 저장본 분석 → 비교 위해 재실행 금지 → 강건 패턴만 신뢰."

**깊은 교훈 (향후 모든 프롬프트 반복에 적용)**: Claude 결정론 불가 + temp 통제 없음
→ **작은 변경의 before/after 인과 측정이 원천적으로 불가**(노이즈와 효과 분리 못 함)
